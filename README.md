## 프로그래밍 언어 기초

- 쉬운언어 선택하기 추천 **JavaScript**, python
- **좀더 욕심 있으면 JAVA**
- 언어에 맞는 편집기 사용, 익숙해지기
- js → vs code
- python → pycham 파이참
- java → inteli J (유료/ 학생버전) , 이클립스 (존나구려)

- atom, //메모장, notepad++

- 자바 공부하면 인프런, 프로그래머스 박재성 강의 듣기((보고 따라 하지말고/ 이렇게 해봐라 시키니까 그거 따라서 만들어보고/ 나중에 결과를 확인))
- 가이드, 로드맵을주고 파트에 맞는 미션 → 미션을 혼자서 (구글링, 책, 지인, 친구) 끝까지 만들어본다 → 결과를 확인
